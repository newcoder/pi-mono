# 每日市场Routine Skill

自动化的盘前盘后市场分析工具，为投资者提供开盘前的市场前瞻和收盘后的交易复盘。

## 触发时机

- **盘前（9:00左右）**：交易日开盘前自动执行，生成盘前简报
- **盘后（15:30左右）**：交易日收盘后自动执行，生成盘后复盘
- **手动触发**：用户输入 `/pre-market` 或 `/post-market` 命令

## 数据来源

| 数据类型 | 来源 | 用途 |
|---------|------|------|
| 宏观数据 | get_macro | 美股/ A50 /汇率 |
| 市场新闻 | get_market_news | 政策/宏观/国际要闻 |
| 板块轮动 | get_sector_rotation | 热门/冷门板块排行 |
| 个股行情 | get_quote | 关注股票表现 |
| 个股新闻 | get_stock_news | 持仓股重大事件 |

## 盘前简报框架（Pre-market）

### 1. 隔夜外围市场回顾
- 美股三大指数（纳指/标普/道指）涨跌幅
- A50期指走势
- 人民币汇率
- 原油价格

### 2. 重大资讯与市场情绪
- 最近24小时政策/宏观/国际新闻汇总
- 情绪判断：偏暖/偏空/中性
- 关键事件：央行决议、重要数据发布、地缘政治

### 3. 板块与行业影响
- 基于隔夜数据和新闻，判断受益/承压板块
- 参考板块轮动热力（如数据可用）
- 行业催化剂：政策利好、产品涨价、订单公告

### 4. A股开盘前瞻
- 高开/低开/平开/震荡判断及依据
- 量能预期

### 5. 关注股票简析
- 逐只分析关注股票的开盘预期
- 引用实际价格数据（前收、52周区间）
- 是否有新闻事件影响

## 盘后复盘框架（Post-market）

### 1. 今日市场回顾
- 主要指数涨跌（上证/深证/创业板）
- 领涨/领跌板块排行
- 涨跌家数比
- 成交额变化

### 2. 重大资讯影响评估
- 回顾今日重要新闻对市场的实际影响
- 验证盘前预判是否准确
- 是否有意外事件

### 3. 关注股票表现
- 逐只分析 watchlist 股票的涨跌原因
- 引用收盘价、涨跌幅、成交额
- 与大盘/行业对比

### 4. 关键决策验证
- 对照今日实际走势验证盘前建议
- 判断建议是否合理（命中/偏差/错误）
- 偏差原因分析

### 5. 问题与反思
- 分析中存在的偏差、遗漏或过度推断
- 模型自身局限（如数据时效、新闻覆盖不全）

### 6. 明日关注计划
- 需要跟踪的具体事项（价格点位、成交量）
- 潜在风险事件（解禁、业绩公告、政策发布）
- 新闻同步提醒（哪些股票需要同步新闻）

## 数据同步建议

为获得最佳分析效果，建议在日常使用中保持以下数据同步：

```bash
# 盘前/盘后执行前，先同步市场新闻
python ~/.agents/skills/a-share-analysis/scripts/market_news_sync.py --sources cls --limit 100

# 同步关注股票的新闻
python ~/.agents/skills/a-share-analysis/scripts/news_sync.py --code 600519 --limit 10
```

## 输出规范

- 盘前简报：控制在 800 字以内，结构清晰
- 盘后复盘：控制在 800 字以内，数据驱动
- 所有涨跌数据保留2位小数
- 所有引用数据必须来自工具返回结果，禁止编造
- 偏差和错误需如实记录，不回避

## 免责声明

所有分析仅基于公开数据，不构成投资建议。市场有风险，投资需谨慎。
