# 股票分析提示词框架（完整版）

以下是一套系统化的股票分析提示词框架，涵盖基本面、技术面、资金面、机构观点、行业竞争格局等维度，可根据分析需求选择性使用。

**分析要求**：
1. 使用可获得的最新数据
2. 每个分析维度请给出明确的结论和依据
3. 保持客观中立，多空因素均需呈现
4. 最终给出清晰的操作建议
5. 如果引用了研报，一定要注意研报的发布日期，第一，发布日期离现在不能超过三个月；
   第二，当时的股价和现在的股价如果差距过大，这份研报就没有参考价值了，不能使用
6. 输出报告必须包含信息：
   - 股票名称
   - 股票代码
   - 分析日期
   - 每个分析维度的结论和依据
   - 操作建议

## 使用说明

1. **按需裁剪**：可根据分析目的选择部分模块，不必全部使用
2. **更新时效**：确保所有数据引用最新日期
3. **多空平衡**：避免选择性呈现信息，保持客观
4. **结论明确**：最终给出清晰的操作建议，避免模棱两可
5. **风险充分**：风险提示需具体，不流于形式
6. **新增模块说明**：第二部分“行业与竞争格局分析”专门解答“行业空间、竞争优势、份额能否保住”三大核心问题，是判断公司长期投资价值的关键前置分析
